# Python学生管理系统 python369

## 项目概述

本项目是一款基于Python的学生管理系统，旨在帮助学校或教育机构高效管理学生信息、课程安排及成绩记录。

## 技术栈

- 后端：Python 3.8+
- 前端：HTML, CSS, JavaScript
- 数据库：MySQL
- 框架：Flask

## 功能模块

### 学生信息管理

- 学生信息录入
- 学生信息查询
- 学生信息修改
- 学生信息删除

### 课程管理

- 课程信息录入
- 课程信息查询
- 课程信息修改
- 课程信息删除

### 成绩管理

- 成绩录入
- 成绩查询
- 成绩修改
- 成绩删除

## 系统角色

- 管理员：负责维护系统及用户管理
- 教师：负责录入、查询、修改学生成绩
- 学生：查询个人信息及成绩

## 运行环境

- 操作系统：Linux或Windows
- Python版本：3.8+
- 数据库：MySQL 5.7+

## 部署步骤

1. 安装Python 3.8+
2. 安装MySQL数据库
3. 克隆代码仓库
4. 安装依赖项：pip install -r requirements.txt
5. 初始化数据库：python manage.py db init
6. 运行应用：python manage.py run

- 界面简洁，操作简便
- 基于Python，易于维护与扩展
